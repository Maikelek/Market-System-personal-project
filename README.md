#  Simple market system
## Made as personal project for my minecraft server 
#

Every user can add an advertisement for his/hers items. Other users are able to see his advertisement and reserve the item. Everyone can see who reserved this item and nobody can reserve it anymore. 

### Used technologies
#
Made with **LAMP STACK**
| Technologies used
| -------------    
| Linux (Hosting)     
| Apache (Web server)            
| MySQL/MariaDB (Database)
| Php (Programming language)
| Bootstrap (CSS framework)
